# 🌌 Latent Space Explorer

A powerful and interactive 2D/3D visualization tool for exploring Word Embeddings (GloVe) using Principal Component Analysis (PCA).

## ✨ Features
- **Interactive 3D/2D Visualizations:** Explore latent spaces with real-time camera rotation (Pitch & Yaw) or switch to a clean flat view.
- **Word Arithmetic Lab:** Perform semantic equations (e.g., *King - Man + Woman = Queen*).
- **Distance Metrics:** Calculate distances between words using interchangeable algorithms (Euclidean & Cosine).
- **K-Nearest Neighbors (KNN):** Find and highlight the closest semantic neighbors for any given word.
- **Cluster Centroids:** Calculate the mathematical center point of multiple concepts.
- **Robust Undo System:** Revert your last actions seamlessly.

## 🏗️ Design Patterns Implemented
This project heavily relies on Object-Oriented Programming (OOP) best practices and GoF Design Patterns:
- **Command Pattern:** Powers the Undo/Redo mechanism.
- **Strategy Pattern:** Enables switching between distance calculation metrics at runtime.
- **Observer Pattern:** Synchronizes the mathematical Engine state with the visual GUI.
- **Decorator Pattern:** Manages visual highlights (e.g., specific colors for selected words and neighbors).

## 🛠️ Tech Stack
- **Backend/Engine:** Java
- **GUI:** Java Swing
- **Data Processing:** Python (scikit-learn, gensim) for PCA dimensionality reduction.
- **JSON Parsing:** Google Gson

## 🚀 How to Run Locally
1. Clone the repository to your local machine.
2. Open the project in IntelliJ IDEA.
3. **Important:** Open the `lib` folder, right-click on `gson-2.10.1.jar`, and select **"Add as Library..."**.
4. Run `Main.java` located in `src/engine/`.

*(Note: The JSON vector data files are pre-generated by the included Python script for instant loading).*
